A better scheduling tool for SCU students.

API for managing user information on the SCU Schedule Helper Chrome extension.

Authentication

Get an access token (JWT)

Get a new JWT for the current user, using a Google OAuth token, or a refresh token.

Authorizations:
oauthbearerRefreshToken

Responses

Production server

https://api.scu-schedule-helper.me/authToken

Response samples

  • 200
  • 401
  • 500
Content type
application/json
Example
{
  • "refreshToken": "string",
  • "oAuthInfo": {
    • "email": "string",
    • "name": "string",
    • "picture": "string"
    },
  • "accessToken": "string",
  • "accessTokenExpiration": "2019-08-24T14:15:22Z"
}

Users

Create a new user

Create a new user with the specified information.

Authorizations:
bearerAccessToken
Request Body schema: application/json
required
name
required
string

The user's full name.
subscription
required
string

A stringified webpush subscription object (so that they can receive notifications from our server).

photo
string

A base64-encoded image of the user's profile picture.

photoUrl
string

A URL to the user's profile picture.

Responses

Production server

https://api.scu-schedule-helper.me/users

Request samples

  • Payload
Content type
application/json
{
  • "name": "John Doe",
  • "subscription": "{ endpoint: \"https://fcm.googleapis.com/fcm/send/id\", expirationTime: null, keys: { p256dh: \"\", auth: \"\", }, };",
  • "photo": "string",
  • "photoUrl": "string"
}

Response samples

  • 400
  • 401
  • 500
Content type
application/json
{
  • "message": "Error message."
}

Update a user

Authorizations:
bearerAccessToken
path Parameters
userId
required
string

The ID of the user to update (for the contextual user, this should just be "me").

Request Body schema: application/json
required
Any of
add
Array of strings

Items to add to the list.
remove
Array of strings

Items to remove from the list.

Responses
